# Hello Rails React App

The Hello Rails React App is a simple app that integrates React into Rails and shows one of five random messages on the screen.

## Built With

- ![Ruby](https://img.shields.io/badge/Ruby-20232A?style=for-the-badge&logo=ruby&logoColor=61DAFB)

## Getting Started

### Installation

1. Clone the repo

   ```sh
git clone git@github.com:the-catalystmc/budget-app.git   
    ```

### Usage

1. Install ruby gems dependencies

   ```sh
   bundle install
   ```

2. Install node dependencies

   ```sh
   npm install
   ```

3. Database creation

   ```
   rails db:setup
   ```

4. Run the web server

   ```sh
   rails server
   ```

5. Access to the website in a browser with the following link

   ```sh
   localhost:3000
   ```
